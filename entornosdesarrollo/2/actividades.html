Obtener el código objeto a partir del código fuente de los 3 archivos siguientes: ```c //------------- // datos.c //------------- char *mensaje="Hola a todos y todas"; int num1 = 8; int num2 = 10; ``` ```c //------------- // suma.c //------------- int suma (int a, int b) { return a + b; } ``` ```c //------------- // main.c //------------- #include <stdio.h> int suma (int a, int b); extern char *mensaje; extern int num1, num2; int main(){ printf("%s\n", mensaje); printf("%d\n", suma (num1, num2) ); return 0; } ``` ```bash # Para obtener código objeto gcc -c main.c datos.c suma.c ``` Deberemos obtener 3 archivos: `main.o`, `suma.o` y `datos.o` <hr> 1. Obtener el código binario ejecutable a partir del código objeto de los 3 archivos anteriores: ```bash # Para obtener código objeto gcc -o programa main.o datos.o suma.o ``` Deberememos obtener un archivo `programa` binario ejecutable.
